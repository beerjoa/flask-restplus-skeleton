### INSTALL

    1. set python virtual environment

        Make sure you have `pyenv` and `pyenv-virtualenv` installed beforehand
        
        On a Mac: 
            $ make Darwin
        On a Linux: 
            $ make Linux

    2. build the virtual environment and install requirements

        $ make build
